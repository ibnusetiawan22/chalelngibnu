Tentang Saya

Junior Web Developer

Saya adalah mahasiswa D3 Teknologi Komputer Universitas Bina Sarana Informatika semester 4. Saya telah mempelajari berbagai aspek teknologi komputer seperti pemrograman, jaringan, sistem operasi, dan Desain Grafis. Saya memiliki pengalaman magang 6 bulan di Aldi Games Playstation, di mana saya mengelola server game, membantu pemain, dan menyelesaikan masalah teknis. Pengalaman ini menggabungkan teori dengan praktik dan membantu saya belajar, berkembang terhadap teknologi sekarang.

Saat ini, saya juga mengikuti program MSIB Magang Studi Independen di Kampus Merdeka bersama LearningX. Program ini memungkinkan saya untuk mendapatkan pengalaman kerja yang lebih real-world dan memperdalam pengetahuan saya di bidang teknologi komputer. Saya yakin dengan kombinasi pendidikan formal, pengalaman magang, dan program MSIB, saya akan siap untuk memasuki dunia kerja dan memberikan kontribusi yang positif di bidang teknologi.
